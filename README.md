Companion addon to [ZoneTimer](https://www.curseforge.com/wow/addons/zonetimer). Displays a per-zone tally of total time and gold in a single window.

Usage:

```
/ztt
```

Streamer and youtuber [@PathalerixGold](https://www.youtube.com/@PathalerixGold) said he wanted the ZoneTImer addon to get an update to display this data as an excel sheet. I was bored so I did it myself.
